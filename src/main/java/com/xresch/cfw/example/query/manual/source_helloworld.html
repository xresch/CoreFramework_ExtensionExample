The helloworld source creates randomized data mainly related to hello world topics! Makes hello world great again!

The following query creates 42 hello world records:

| source helloworld

Following query sets the output to 420 records, sets the range for lucky_number and the name to 'Jane Doe' for the hi_there message:

| source helloworld 
	records=420 
	min=40 
	max=50 
	name="Jane Doe"
